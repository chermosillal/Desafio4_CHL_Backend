# Frontend - Desafío 4

Aplicación React para la gestión de posts con sistema de likes.

## Instalación

```bash
npm install
```

## Ejecución

### Modo desarrollo
```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

### Producción
```bash
npm run build
npm run preview
```

## Configuración

Asegúrate de que el backend esté corriendo en `http://localhost:3000` antes de iniciar el frontend.

## Características

- Crear nuevos posts con título, imagen y descripción
- Visualizar todos los posts
- Sistema de likes (próximamente)
- Eliminar posts (próximamente)
